¿Qué es JSON?

JSON (JavaScript Object Notation) es una forma sencilla de almacenar y compartir datos. Piensa en él como una hoja de cálculo donde guardas información con etiquetas claras y valores.

Ejemplo Real: Inventario de Oficina

      
Imagina que administras el inventario de una oficina. Así se ve en JSON:

      
{
  "inventario": [
    { "item": "Silla", "cantidad": 20 },
    { "item": "Escritorio", "cantidad": 10 },
    { "item": "Portátil", "cantidad": 5 }
  ]
}
      

      
Este formato permite intercambiar datos fácilmente entre sistemas.
