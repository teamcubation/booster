Teamcubation — Paso a paso: Solicitar API Key Gmail para conectar con n8n
Cómo solicitar una API Key de Gmail para n8n
1
Ingresá a Google Cloud Console
Accedé a Google Cloud Console usando tu cuenta de Google.
2
Creá un proyecto nuevo
Hacé clic sobre el selector de proyecto arriba a la izquierda y elegí “Nuevo Proyecto”. Poné un nombre descriptivo (por ejemplo, "n8n Gmail") y presioná Crear.
3
Habilitá la API de Gmail
Dentro del proyecto, buscá “Gmail API” usando la barra de búsqueda de la consola. Seleccioná Gmail API y hacé clic en Habilitar.
4
Configurá la pantalla de consentimiento OAuth
En el panel izquierdo, buscá “Pantalla de consentimiento OAuth”. Configurala como "Externa", completá los campos requeridos y guardá. (No es necesario enviar para verificación si solo lo usás personalmente.)
Tip: completá el mail de soporte y el nombre de app como "n8n Gmail".
5
Creá credenciales OAuth 2.0
En “Credenciales” > “+ Crear credenciales” elegí Cliente de OAuth. Seleccioná "Aplicación web" y agregá http://localhost como origen autorizado.
En la sección de URIs de redireccionamiento, agregá:
  • http://localhost:5678/rest/oauth2-credential/callback
Luego presioná Crear.
Guardá el ID de cliente y el Secreto. Los vas a necesitar en n8n.
6
Ingresá credenciales en n8n
Entrá a n8n, creá una nueva credencial de tipo Gmail OAuth2 API e ingresá el Client ID y Client Secret copiados. ¡Listo!
Te recomendamos probar la conexión ¡y ya podés automatizar Gmail!
